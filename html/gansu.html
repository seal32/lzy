甘肃10+日自由行攻略（广州出发）

行程亮点：融合河西走廊的历史文化与甘南的自然风光，穿插经典徒步路线，体验西北美食与多元地貌。

推荐季节：6-10月（避开冬季严寒，夏季草原繁茂，秋季色彩丰富） 。

行程概览 路线规划：兰州 → 张掖 → 嘉峪关 → 敦煌 → 夏河（甘南） → 扎尕那 → 兰州
交通方式：飞机+高铁+包车/拼车（灵活高效，适合自由行） 。

D1：广州 → 兰州 交通：广州直飞兰州（约3.5小时）。兰州中川机场到市区非常远，除了跟随航班发车的机场大巴还可以坐城际高铁抵达市区，运营时间是从早上6点左右到晚上10点左右。 机场巴士1号线（城关区/火车站方向，中川机场 → 万达十字 → 东方大酒店 → 兰州火车站，30分钟每班，30元；机场巴士2号线（西关什字/黄河风情线方向，中川机场 → 恒大绿洲 → 大沙坪小学 → 庙滩子消防中队 → 中山桥南 → 新胜利宾馆）；深夜抵达（23:00后） → 优先选3号线大巴（运营至凌晨1点），大巴售票点在T1/T2航站楼到达厅。城际列车（机场-市内各火车站，20元）；网约车和出租车（约100-120元）。

住宿：格林豪泰酒店（西关十字中山桥智选店，中山路156号）、速8精选酒店（小西湖店）、豪爵宾馆（瑞德摩尔店）、天星快捷连锁旅馆。

景点：兰州黄河风情线：中山桥、黄河母亲雕像、水车博览园，甘肃省博物馆、白塔山公园，傍晚乘游轮赏夜景。

D2：兰州 → 张掖。 交通：兰州西站乘高铁至张掖（3小时）。 住宿：张掖市区经济型酒店（如张掖丝路之星酒店，约200元/晚）。
美食：牛肉小饭、臊子面、搓鱼子。
景点：张掖七彩丹霞（日落时分最佳）； 大佛寺；平山湖大峡谷（徒步20多公里，探秘“中国的科罗拉多”峡谷奇观。徒步路线有“一线天”、“百步峡”和“灵芝谷”）。

D3：张掖 → 嘉峪关 交通：高铁（1.5小时）或拼车（2.5小时）。
住宿：嘉峪关市区经济型酒店（如嘉峪关长城宾馆，约180元/晚）。
美食：嘉峪关烤肉、烤全羊。
景点：嘉峪关关城（明代长城西端起点，感受边塞雄风）。

Day 4-5：嘉峪关 → 敦煌 交通：火车（4-5小时）或拼车（5小时）。
住宿：优先位置选：河岸听风青旅、锦江之星品尚。亲近自然选：沙景源客栈（沙漠活动）、柒月客栈（农家风情）。
美食：驴肉黄面、杏皮水、胡羊焖饼。
景点： 1、莫高窟：开放时间12/01-03/31 09:00-17:30；04/01-11/30 08:00-18:00，提前3-5天预约。A 类票238元可参观 8 个洞窟 + 数字电影；B 类票100元可参观 4 个洞窟；一般A类票很难预约到，大多是B类。乘坐摆渡车前往洞窟区，观赏千年佛教壁画与雕塑（九层楼、三层楼、藏经洞……），洞窟内禁止拍照。
2、鸣沙山月牙泉：开放时间07:30~18:00，微信公众号提前预约，门票110元。中门入， 线路一：游客中心—鸣月广场东—陶瓷公路—药王庙遗址—小泉湾广场—鸣月阁—月牙泉—登山。全程步行需要3个小时左右，乘坐观光车需要2.5-3.5个小时。 3、玉门关：感受“春风不度玉门关”的苍凉。 4、雅丹地质公园：观看雅丹地质公园雅丹魔鬼城日落美景。著名景点有：金狮迎宾、狮身人面、孔雀玉立、西海舰队、神龟出海、雅丹公主、伟人像、雅丹城堡、桃园结义、天外来客……等等 。分为北线和南线两个区域，游客可以选择乘区间车游玩北线，全程约2小时。

Day 6-7：敦煌 → 夏河（甘南）
交通：敦煌飞兰州（1.5小时），兰州汽车南站乘大巴至夏河，8:30、10:30、14:00、15:00、16:00，票价为75元（3.5小时）。夏河县没有公交车，直接打车，县内均为3元/人。
住宿：雪山客栈、夏河霞客行青年旅舍……
美食：藏式糌粑、牦牛酸奶、手抓羊肉。
景点： 1、拉卜楞寺：夏河县公共型汽车站出来后徒步前往该寺庙。世界藏学府，转经长廊祈福，感受藏传佛教氛围。（门票20元/人）。 2、桑科草原： 从拉卜楞寺西口出来后，对面停车场有面包车（5元/人）可以前往桑科草原。如果不想打车，可以选择徒步上山，来回往返共计徒步2.5小时。非常适合拍照。 3、甘加秘境：八角古城，草原、白石崖，作海寺。人少景美的隐世之地。

Day 8-9：夏河 → 扎尕那（徒步核心） 交通：先从夏河坐车到合作市（很多班次，1.5小时），从合作客运中心乘坐前往迭部县的车（只有上午有班次），或直接包车前往扎尕那（约4小时）。
（或选择先去朗木寺镇住一晚，游览朗木寺、徒步纳摩大峡谷。） 住宿：扎尕那藏式民宿（如扎尕那云居客栈，约200元/晚）。
徒步路线：扎尕那小环线（5公里）、 扎尕那涅甘达哇神山环线（OCM地形图有标注，全程约20公里）。
亮点：高山草甸、峡谷森林、藏寨风情，秋季层林尽染，夏季油菜花海。
贴士：建议雇佣向导（约300元/天），携带防雨装备和高热量零食。

Day 10：扎尕那 → 兰州 → 广州 交通：包车至兰州（8小时），或扎尕那→合作市→兰州（大巴+高铁）。
返程：兰州中川机场直飞广州。

预算参考 交通：机票（往返约2500元）+ 高铁/包车（约1500元）。
住宿：经济型酒店/青旅（约150-200元/晚，总计约1800元）。
门票+徒步：约800元（莫高窟238元，七彩丹霞75元，扎尕那徒步向导费）。
餐饮：日均100元，总计约1000元。
总计：人均约7000-8000元（不含购物）。

注意事项

高原反应：甘南平均海拔3000米，避免剧烈运动，备氧气瓶。

防晒防沙：敦煌沙漠紫外线强，需防晒霜、墨镜、防风沙头巾。

文化尊重：寺庙内禁止拍照，勿触碰转经筒。

美食地图
兰州：牛肉面、甜胚子、灰豆子。
敦煌：驴肉黄面、杏皮水。
甘南：牦牛肉火锅、酥油茶。

徒步装备清单
必备：登山鞋、冲锋衣、防晒帽、水壶、急救包。
选带：登山杖、头灯、轻便雨衣。

此攻略综合了河西走廊的历史文化与甘南的自然野性，兼顾经典景点与小众体验，适合追求多样化的自由行游客。
